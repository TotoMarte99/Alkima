Diseño y
Desarrollo Web

Nos ponemos en tu piel para ofrecerte un diseño web personalizado. Porque el buen diseño y desarrollo web es aquel que permite la interacción entre marca y consumidores, ayudando a convertir leads en compradores.

Un diseño web adaptado a tus necesidades

Realizar el sitio web adecuado requiere un compromiso y arduo labor, en Alkima somos concientes de esto, es por eso que, cuando realizamos un diseño web lo hacemos a medida, ya que nuestro objetivo es que cada usuario que ingrese al sitio web sienta que adquiere valor de calidad.
De esta manera podremos maximizar las conversiones de tu proyecto web.

¿Como empezamos?

Antes de comenzar a desarrollar el sitio web debemos tener claro 3 factores


  • Cuales son las metas del negocio

  • A quien nos estamos dirigiendo

  • De que forma los potenciales usuarios consumen la información

¿Como te ayudamos?

La creación de un sitio web escalable es una tarea compleja. No basta con diseñar paginas web esteticas, hay que tambien poner foco en lograr webs rapidas, intuitivas, comunicativas y con buena retención de los usuarios.

Para lograr esto te ayudamos en:


  • Diseño de sitios web corporativos, blogs o tiendas online

  • Consultorias y desarrollo se sitios web adaptativos

  • Mantenimiento web

  • Remodelación de sitios web

  • Creacion de contenido y administracion de redes sociales

¿Como es el proceso de desarrollo?


1.Planificación
y Diseño

Nos contactamos con el cliente para saber que requiere su proyecto(hosting,dominio, diseños y formato de las paginas web, actualizaciónes, etc.

2.Recepcion de
Material

Solicitamos al cliente los materiales posibles tales como (logotipos, fotografias, videos, textos, colores, entre otros), para poder comenzar con el diagrama del sitio web

3.Estructuración
de la web

Definimos el estilo, tipografia, colores, cantidad de páginas/secciónes, menús, todo lo necesario para crear una web original y estetica.

4.Gestor
de contenidos

En caso que el proyecto a realizar la instalacion de un gestor de contenidos (CMS), se procesera a realizar las intalaciónes correspondientes.

5.Presentación

Nos reunimos con el cliente de forma online o presencial para presentar el diseño final y hacer revisiones y/o modificaciónes necesarias, en caso de que el cliente las solicite. Para asi concretar la etapa de diseño y comenzar la etapa de desarrollo

6.Desarrollo

Una vez el cliente está conforme con el diseño o en caso de utilizar worpress, con el tema seleccionado, comenzamos a desarrollar todas las funciones web, previo a un pago incial de 25%% del total del valor en condición de seña.

7.Revisiones y Correciones

Previo al lanzamiento del sitio web, luego de una exaustiva revision por parte de nuestro personal. Coordinaremos una nueva reunion con el cliente para que visualice todo el desarrollo y si lo ve necesario nos solicite modificaciónes u arreglos.

8.Soporte

En caso de implementar un CMS (gestor de contenidos) en tu pagina web, te enseñaremos a uttilizar las funciones del mismo de la manera más intutiva, para que puedas tener el control perfecto sobre el gestor en cuestión.

9.Lanzamiento

Una vez finalizadas todas las tareas correspondientes con el diseño y creación del sitio web y previo pago total por los servicios prestados por parte del equipo de ALKIMA, publicaremos el sitio web.
